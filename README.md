# Twobit

Ruby bindings to lib2bit.

* https://github.com/deeptools/py2bit
* https://github.com/dpryan79/lib2bit

## Installation

```sh
gem install twobit
```

## Usage

```
Twobit.new("foo.2bit")
Twobit.close
```
